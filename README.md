# Z00

# 概要
基於 PHP 和 MySQL 開發，旨在幫助管理人員追蹤和管理動物園內的動物及訪客數據。系統提供動物數量統計、成人和兒童訪客的數據分析，並能顯示訪客數量，分別統計總數、今天和昨天的訪客情況。所有數據會實時更新並顯示在管理界面，方便管理員即時掌握動物園的運營狀況。
